Add some effects to any Gutenberg block using Anime.js

== Installation ==
1. Download the ZIP archive and upload it through *Plugins â†’ Add New* in your WordPress admin.
2. Activate the plugin.

== Usage ==
- Create or edit a post or page.
- Select any block, then open the **Anime.js Animation** panel in the sidebar.
- Pick an effect and save. Anime.js will be loaded automatically on that page only.

== Uninstall ==
The *uninstall.php* file removes the extra attributes and the `_gaa_has_animation` meta key.
